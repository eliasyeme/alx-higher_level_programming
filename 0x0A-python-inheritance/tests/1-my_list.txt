=======================
How to Use 1-my_list.py
=======================

Passing case
============

Import
::
	>>> MyList = __import__('1-my_list').MyList
	>>>

Base case
::
	>>> my_list = MyList()
	>>> my_list += [2, 4, 1, 5, 3]
	>>> my_list
	[2, 4, 1, 5, 3]
	>>> my_list.print_sorted()
	[1, 2, 3, 4, 5]
	>>> my_list
	[2, 4, 1, 5, 3]

::
	>>> my_list = MyList([2, 7, 5, 1])
	>>> my_list.print_sorted()
	[1, 2, 5, 7]

::
	>>> my_list = MyList([])
	>>> my_list.print_sorted()
	[]

::
	>>> my_list = MyList()
	>>> my_list.append(0)
	>>> my_list.print_sorted()
	[0]
	>>> my_list
	[0]

::
	>>> my_list = MyList()
	>>> my_list += [0, 0, 0, 0]
	>>> my_list.print_sorted()
	[0, 0, 0, 0]

::
	>>> MyList = __import__('1-my_list').MyList
	>>> my_list = MyList()
	>>> my_list += [2, -1, -2, 1, 0]
	>>> my_list.print_sorted()
	[-2, -1, 0, 1, 2]

Error case
==========

Not defined object
::
	>>> MyList = __import__('1-my_list').MyList
	>>> my_list2.print_sorted()
	Traceback (most recent call last):
	NameError: name 'my_list2' is not defined


Parent object
::
	>>> my_list = [1, 2, 3]
	>>> my_list.print_sorted()
	Traceback (most recent call last):
	AttributeError: 'list' object has no attribute 'print_sorted'
